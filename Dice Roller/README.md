#  Dice Roller App

A simple Android app built using Java and Android Studio.

## Features
- Roll a six-sided dice
- Random number generation
- Dice image changes dynamically
- Dice rolling sound effect

## Technologies Used
- Java
- Android Studio
- XML Layouts
- MediaPlayer

## What I Learned
- Event handling using OnClickListener
- Random number generation
- Updating ImageView and TextView
- Working with drawable and raw resources
